# 🚀 Web Showroom - Prototipos en Vivo

Bienvenido a mi repositorio de exhibición. Aquí centralizo prototipos de páginas web y conceptos de diseño listos para ser visualizados en el navegador por clientes y colaboradores.

## 📂 Estructura del Proyecto
Cada carpeta en la raíz representa un prototipo independiente con su propio `index.html`.

- `/prototipo-empresa-x`: Propuesta de landing page para sector corporativo.
- `/prototipo-ecommerce`: Concepto de tienda minimalista.
- `/index.html`: Galería principal de acceso a todos los proyectos.

## 🌐 Visualización Directa
Todos los prototipos están desplegados mediante **GitHub Pages**. Puedes acceder a la galería principal en:
